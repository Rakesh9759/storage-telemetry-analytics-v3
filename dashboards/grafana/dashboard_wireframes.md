# Grafana Dashboard Wireframes

## Dashboard 1: Device Health Monitor
Panels:
- total_iops over time by device
- avg_latency_ms over time by device
- util_pct over time by device
- aqu_sz over time by device
- anomaly_flag timeline

## Dashboard 2: Anomaly Operations View
Panels:
- count of anomalies over time
- count of critical anomalies over time
- affected devices table
- latest anomaly events table

## Suggested filters
- device
- time range
- severity
